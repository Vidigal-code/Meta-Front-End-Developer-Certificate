### You want to insert a paragraph on your website. Choose the correct format for the opening and closing paragraph tags:

- [ ] Opening tag: `<p>` Closing tag: `<p/>`.
- [ ] Opening tag: `<p>` Closing tag: `<p>`.
- [x] Opening tag: `<p>` Closing tag: `</p>`.
